### Installation

Clone the repo, install the dependencies and start the server. This was tested only in Chrome.

```sh
$ git clone https://github.com/throwaway1920/breather-challenge.git
$ cd breather-challenge
$ npm install
$ npm start
```

### Dependencies

```sh
* react 16.8.6
* axios 0.18.0
* redux: 4.0.1
* redux-saga: 1.0.2
* styled-components: 4.2.0
```
